Welcome to Cold Sweat's documentation!
===================================

**Cold Sweat** is a mod for Minecraft that adds a dynamic temperature system to the game.<br>
This documentation details how third-party mod developers can hook into Cold Sweat's systems and use them to add/change content, implement compatibility for other mods, etc.<br>
<br>
This documentation assumes knowledge of Java and Minecraft/Forge.<br>
https://mcforge.readthedocs.io<br>
https://forge.gemwire.uk/wiki/Main_Page

.. note::

   This project is under active development.
